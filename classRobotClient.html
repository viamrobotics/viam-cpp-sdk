Detailed Description

gRPC client for a robot. This class should be used for all interactions with a robot.

Create a robot client that is connected to the robot at the provided local socket. Creates a direct connection to the robot using the unix:// scheme. Only useful for connecting to robots across Unix sockets.

Gets a shared_ptr to a Resource Base.

Args: name: the name of the resource

Raises: Throws an error if the requested resource doesn't exist or is of the wrong type.

This method should not be called directly except in specific cases. The type-annotated resource_by_name<T>(name) overload is the preferred method for obtaining resources.

Because the return type here is a ResourceBase, the user will need to manually cast to the desired type.
